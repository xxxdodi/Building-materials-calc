# Строительный калькулятор

Прототип клиентского веб-приложения для автоматизированного расчёта отделочных материалов с учётом геометрии помещений, проёмов и технологических норм расхода.

Проект разработан в рамках проектно-технологической практики и ориентирован на автономную (offline) работу без использования серверной части и внешних API.

---

## Основные возможности

- расчёт пяти типов материалов: плитка, ламинат, обои, краска, штукатурка;
- динамические формы ввода в зависимости от выбранного материала;
- учёт дверных и оконных проёмов с вычитанием площади;
- материал-специфические алгоритмы (швы, раппорт, расход, запасы);
- автоматическое сохранение истории расчётов в LocalStorage;
- адаптивный интерфейс для мобильных и десктопных устройств;
- формирование детализированной сметы с выводом на печать.

---

## Архитектура приложения

Приложение реализовано как одностраничное веб-приложение (SPA) с паттерном проектирования «Стратегия» для организации расчётной логики:

- **HTML5** — семантическая разметка, валидация форм;
- **CSS3** — адаптивная сетка (Grid, Flexbox), CSS Variables;
- **JavaScript (ES6+)** — модульная структура, классы, LocalStorage API;
- **Font Awesome** — векторные иконки.

Каждый тип материала представлен отдельным объектом конфигурации с набором полей ввода и методом расчёта, что обеспечивает расширяемость без модификации ядра приложения.

## Алгоритмы расчёта
| Материал   | Основные параметры               | Формула                               |
| ---------- | -------------------------------- | ------------------------------------- |
| Плитка     | размер, шов, запас               | `n = ⌈S / ((a+s)(b+s))⌉ × (1+запас)`  |
| Ламинат    | площадь упаковки, запас          | `n = ⌈S / S_упак⌉ × (1+запас)`        |
| Обои       | раппорт, высота, ширина рулона   | учёт целого числа раппортов на полосу |
| Краска     | расход л/м², слои, объём банки   | `V = S × расход × слои`               |
| Штукатурка | толщина, расход кг/м², вес мешка | `m = S × расход × (толщина/10)`       |
